# kunalvats.github.io
Interactive Power BI dashboard analyzing mobile sales performance, customer trends, and transaction insights.

# 📱 Mobile Sales Dashboard Analysis

## 📌 Objective

Analyze mobile sales data to understand sales performance, customer behavior, payment methods, and brand-wise trends.

---

# 🛠 Tools Used

* Power BI
* Excel

---

# 📊 Dashboard Features

* Total Sales Analysis
* Brand-wise Sales Insights
* Customer Ratings
* Payment Method Analysis
* City-wise Sales Visualization
* Monthly Quantity Trends

---

# 📈 Key Insights

* Total Sales reached 769M
* Apple generated highest sales
* UPI was one of the most used payment methods
* Sales performance varied across cities and months
